As Tags

Tag footer

É a tag que cria um rodapé para sua pagina.
Normalmente um rodapé contém informações sobre o autor da seção de dados, direitos autorais ou links para documentos relacionados.

Tag Head

tag head é a responsavel pelo cabeçalho da pagina.
La estará ad imformções gerais da paginacomo nome do autor,palavra chave e etc

Tag pre

é a tag utilizada para representar texto pré-formatado. Um texto dentro desse elemento é tipicamente exibido em uma fonte não proporcional da mesma maneira em que o texto original foi disposto no arquivo. Espaços em branco são mantidos no texto da mesma forma em que este foi digitado.

Tag code

A tag code e utilizada para apresenta seu conteúdo estilizado de maneira a indicar que o texto é um pequeno fragmento de código. Por padrão

Tag em

marca o texto que tem ênfase. O elemento "em" pode ser alinhado, com cada nível de alinhamento indicando um grau maior de ênfase.

Tag Mark

A tag mark representa um trecho de destaque em um texto, por exemplo, uma sequência de texto marcado como referência, devido à sua relevância em um contexto particular.
Por Exemplo, pode ser utilizado em uma página que mostra os resultados de uma busca onde todas as instâncias da palavra pesquisadas receberam destaque.

Tag strong

O elemento strong é utilizado em conteúdos que são de "grande importância", incluindo coisas de urgentes (como alertas). Podem ser sentenças que são de grande importância para toda a página, ou, você pode meramente tentar pontuar que algumas palavras são de grande importância, comparado ao conteúdo próximo.

Tag u

A tag U ela serve para sublinhar o trecho que esta dentro dele para que destaque certa parte

Tag samp

A tag samp "Sample Text Element" ou exemplo é utilizado para destacar possíveis resultados de algum processo "on line" ou a explicação em algum tutorial em formato pdf, por exemplo.

tag kbd

A Tag KDB "Keyboard Input Element" ou chave(s) de entrada ou comando(s) é utilizado para orientar usuários em instruções normalmente "on line", ou impressas. Para destacar o(s) comando(s) a fonte visualizada é "monospace

Tag section

O elemento HTML section representa uma seção genérica contida em um documento HTML, geralmente com um título, quando não existir um elemento semântico mais específico para representá-lo.
